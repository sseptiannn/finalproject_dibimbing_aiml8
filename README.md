# Customer Segmentation & Default Risk Analysis


This project focuses on:
- Customer segmentation using KMeans clustering
- Default risk prediction using supervised ML
- Business insights from payment behavior


Tech Stack:
- Python, Scikit-learn
- MLflow (experiment tracking)
- Streamlit (demo app)
- Docker (optional deployment)
